transix.abc_to_sym#

transix.abc_to_sym(a, b, c)#

Compute symmetrical components (Fortescue transform) from abc quantities.

Parameters:
acomplex array

Signal a

bcomplex array

Signal b

ccomplex array

Signal c

Returns:
outSequenceABC

Symmetrical components expressed in the abc frame.

out.zerotuple of complex

Zero-sequence components (a0, b0, c0), where a0 == b0 == c0.

out.postuple of complex

Positive-sequence components (a1, b1, c1).

out.negtuple of complex

Negative-sequence components (a2, b2, c2).

Notes

Uses the Fortescue symmetrical component transformation with \(\alpha = e^{j2\pi/3}\).

Examples

>>> import numpy as np
>>> alpha = np.exp(1j * 2*np.pi/3)
>>> abc_to_sym(1, alpha**2, alpha).zero[0]
0j
